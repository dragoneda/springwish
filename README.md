# 新春微信生成器

自动生成个性化的新春拜年微信程序。

## 功能特点

1. **智能关系识别**：自动根据联系人信息判断关系类型（师生、同事、上下级、朋友、家人、同学等）
2. **聊天记录分析**：分析与联系人的聊天记录，提取关键词和重要事项
3. **个性化拜年微信**：根据关系类型和聊天记录，生成声情并茂的拜年微信
4. **用户交互**：征求用户意见，不满意可重新生成
5. **本地数据库**：存储联系人信息和聊天记录

## 安装和使用

### 环境要求

- Python 3.6 及以上版本
- 无额外依赖，使用 Python 内置库和 sqlite3

### 安装

```bash
git clone https://github.com/dragoneda/springwish.git
cd springwish
```

### 运行程序

```bash
python3 index.py
```

### 程序操作

1. 查看联系人列表
2. 添加新联系人
3. 查看聊天记录
4. 添加聊天记录
5. 生成拜年微信
6. 退出程序

### 测试程序

```bash
python3 test.py
```

## 项目结构

- `index.py`：主程序入口
- `database.py`：数据库连接和初始化
- `contact_relation.py`：联系人关系判断
- `chat_manager.py`：聊天记录管理
- `greeting_generator.py`：拜年微信生成
- `user_interaction.py`：用户交互
- `test.py`：测试程序
- `requirements.txt`：依赖库列表
- `data/`：数据库文件存储目录

## 技术原理

1. **关系判断**：基于联系人备注信息进行模式匹配
2. **聊天分析**：提取关键词和重要事项
3. **微信生成**：根据关系类型使用不同的模板生成
4. **数据库**：使用 SQLite 进行本地数据存储
